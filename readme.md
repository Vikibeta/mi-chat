## Developing

<br>
一个移动端的即时聊天应用，使用 Express + Socket.io + Vuejs 构建，目前已完成基础的功能，正在努力开发中。

#### 项目进度
- [x] 登录注册
- [x] 私聊系统
- [x] 好友列表
- [x] 未读消息列表
- [x] 个人资料展示与设置
- [x] 退出登录
- [ ] 群
- [ ] 公共聊天室
- [ ] 表情、图片、录音发送
